How to install in Gmail

  1. Click "Copy HTML for Gmail" above for your signature.
  2. Go to Gmail → Settings → See all settings → General → Signature → Create new.
  3. In the signature editor, click the </> HTML source icon (bottom right of toolbar).
  4. Delete any existing content, then paste the copied HTML.
  5. Click the HTML icon again to close source view, then Save Changes.
  6. Send a test email to verify the car image loads (it's hosted on GitHub).
